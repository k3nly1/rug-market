# rug-market
Цей проєкт представляє повноцінний адаптивний сайт інтернет-магазину килимів. Сайт включає головну сторінку з рекламним блоком, каталогом популярних товарів, інформаційними розділами та контактами компанії. Інтерфейс оптимізований для всіх типів пристроїв, підтримує мобільну навігацію, інтерактивні елементи та зручну структуру для майбутньої інтеграції з backend-системою.
Сайт містить:

повноцінну шапку з брендом, меню навігації, кнопками взаємодії та адаптивним burger-меню;

головний банер з презентацією магазину та закликом до дії;

каталог товарів у вигляді сітки з картками, що включають зображення, інформацію про матеріали, розміри та ціну;

кнопки дій для переходу до детальної інформації або додавання товару в кошик;

розділ «Про нас» з описом діяльності компанії;

підвал із контактними даними, службовими посиланнями та соціальними мережами.

Розмітка виконана з використанням семантичних HTML-тегів, адаптивна стилізація побудована на Flexbox та CSS Grid, а логіка мобільного меню реалізована на JavaScript. Це робить сайт готовим для подальшого розширення, інтеграції з CMS або серверною частиною, а також для використання як комерційний продукт.
Інструкції з запуску
Варіант 1. Запуск локально (без серверів)

Завантажити або склонувати репозиторій.

Відкрити файл index.html у браузері.

Сайт працюватиме коректно, оскільки не потребує збірки чи бекенду.

Варіант 2. Запуск через локальний сервер (рекомендовано для коректної роботи ресурсів)

Встановити будь-який локальний сервер, наприклад:

VS Code + Live Server,

Python SimpleHTTPServer,

Node.js http-server.

Запустити сервер у кореневій директорії проєкту.

Відкрити згенеровану адресу (зазвичай http://localhost:3000 або http://127.0.0.1:5500/index.html).

Варіант 3. Розміщення на GitHub Pages

Завантажити проєкт у Github.

У розділі Settings → Pages обрати гілку main та теку /root.

Після збереження згенерується публічне посилання на сайт.
